# 入职流程（hr-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构 HR 内部资料，只允许 HR 和管理员访问。本文用于回答“入职流程是什么”“试用期目标如何制定”等问题。

## 2. 入职前准备
HR 在候选人接受 offer 后确认入职日期、岗位、直属主管、办公地点和材料清单。入职材料包括身份证明、学历证明、紧急联系人、银行卡或薪酬相关信息的授权采集记录。员工档案只允许 HR 和管理员在授权范围内查看。

## 3. 入职当天流程
HR 完成身份核验、员工档案创建、制度签收、账号申请、办公设备领用和新员工培训安排。行政部协助工位和资产，直属主管确认岗位介绍和首周任务。员工不得直接访问其他部门内部知识库，需按权限申请流程开通。

## 4. 试用期目标
试用期目标由直属主管和员工在首周共同制定，HR 负责提醒和归档。目标应包含岗位产出、协作要求、学习计划和转正评估标准。目标变更需要双方确认并记录原因。

## 5. 可回答的问题
- 入职流程是什么？
- 试用期目标如何制定？
- 入职材料包括什么？
- 员工档案谁可以查看？
- 新员工能不能默认访问全部知识库？
