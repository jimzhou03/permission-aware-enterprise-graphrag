# 星海智造机器人有限公司公开政策与参观指引（v0.7.1 Demo）

## 1. 文档标题
星海智造机器人有限公司（StarSea Robotics Co., Ltd.）公开政策与参观指引。

## 2. 适用范围
适用于访客、潜在客户、合作伙伴、媒体与校园来访团队。本文仅包含可公开传播信息，不构成正式合同、报价单、技术实施承诺或服务等级协议。

## 3. 文档目的
- 统一公司公开介绍口径，避免对外信息不一致。
- 明确访客能看到什么、不能看到什么。
- 提供商务合作与参观流程的公开入口，降低沟通成本。

## 4. 公司公开介绍
星海智造机器人有限公司是一家面向企业、学校、园区、展馆和商业综合体的服务机器人解决方案公司。公司提供从机器人本体、调度平台到现场交付与运维支持的一体化方案，聚焦“可落地、可运维、可持续迭代”的项目交付模式。

公开场景包括：
- 前台咨询与业务引导
- 园区巡检与异常上报
- 展厅迎宾与品牌互动
- 楼宇/医院/酒店内部短途配送

## 5. 公开产品线介绍
- 服务机器人：用于前台咨询、路线引导、常见问题答复。
- 巡检机器人：用于园区巡检、设备点检、异常拍照与上报。
- 迎宾机器人：用于展厅接待、活动签到、品牌讲解。
- 配送机器人：用于楼宇、园区、医院、酒店内部短途配送。

公开版本说明：演示资料中的型号、续航、负载和里程参数为虚构示例，用于演示知识库问答效果。

## 6. 访客可了解的信息
访客可了解：
- 公司业务定位与公开产品线
- 典型应用场景与标准化交付流程
- 公开联系方式、预约参观方式、基础合作流程

访客不可访问：
- 内部报价策略与底价区间
- 客户名单、客户联系人、项目履约细节
- 技术接口密钥、设备密钥、系统凭据
- 售后工单细节与故障处置记录
- 员工制度、薪酬绩效、人事档案

## 7. 商务合作公开流程
1. 提交合作需求：通过公开邮箱或电话登记场景、预算区间、目标上线时间。
2. 初步评估：商务人员在 2 个工作日内完成场景可行性初评。
3. 方案沟通：确认需求边界后安排线上或线下方案会。
4. 试点建议：对复杂场景建议先做小范围试点。
5. 正式商务阶段：签署保密与商务文件后进入详细报价和实施排期。

## 8. 参观须知（公开）
- 参观实行预约制，建议至少提前 2 个工作日登记。
- 来访当天需完成访客登记并佩戴访客证。
- 未经授权不得拍摄内部屏幕、流程看板和员工工位资料。
- 演示区可拍照，涉及内部系统页面需现场确认后再拍摄。

## 9. 常见问题（FAQ）
Q1：你们是否提供定制开发？
A：提供，通常先走标准方案评估，再判断是否进入定制阶段。

Q2：是否支持跨城市部署？
A：支持，具体实施计划在商务与交付评估后确认。

Q3：是否可直接获取 API 文档和技术密钥？
A：公开阶段不提供。进入授权合作流程后按权限分级开放。

## 10. 禁止事项与权限边界
- 公开接待阶段不讨论内部底价、客户隐私、员工信息。
- 未进入授权流程前，不开放内部技术文档、密钥和工单细节。
- 所有权限由公司后端系统按角色和部门统一控制。

## 11. 示例场景
- 校园访客提问“公司是做什么的？”：可回答公司定位、产品线与公开场景。
- 访客提问“销售底价是多少？”：应提示无权限并引导走正式合作流程。
- 合作伙伴提问“如何预约参观？”：可提供公开联系方式与预约步骤。
