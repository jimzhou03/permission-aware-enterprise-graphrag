# 采购申请流程（admin-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构行政部内部资料，只允许行政部和管理员访问。本文用于回答“行政部采购流程是什么”“办公用品申请流程是什么”等问题。

## 2. 采购申请
行政部采购流程从业务部门提交采购申请开始。申请人需写明物品名称、数量、用途、预算、期望到货时间和是否涉及项目交付。行政部检查库存，若可调拨则优先调拨；确需采购时进入预算确认。

## 3. 预算与比价
行政部和财务共同确认预算可用性。常规办公用品可按框架供应商采购；金额较高或非标准物品需要至少两到三家供应商比价。比价记录应包含价格、交期、售后、质量风险和推荐理由。

## 4. 审批与入库
采购达到阈值时进入多级审批。审批通过后行政部下单、跟踪到货、验收数量和质量，并完成资产登记或用品入库。申请人领取时签收，后续按资产或低值耗材规则管理。
