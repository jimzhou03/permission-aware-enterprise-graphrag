# 办公资产管理（admin-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构行政部内部资料，只允许行政部和管理员访问。本文用于回答“办公资产如何管理”“设备借用流程是什么”等问题。

## 2. 资产登记
行政部办公资产包括 IT 设备、办公家具、演示设备、会议设备和低值易耗品。采购入库后必须生成资产编号，记录品类、规格、责任人、存放地点、采购批次和状态。未登记资产不得直接发放。

## 3. 领用与借用
员工领用或借用设备需在系统提交申请，说明用途、使用期限和归还时间。行政部检查库存和审批状态后发放，并更新责任人。跨部门借用演示设备时，行政部按业务优先级协调，不通过口头私借。

## 4. 盘点与归还
行政部定期盘点资产，核对账实差异、损坏状态和闲置情况。归还时检查外观、配件和使用记录。丢失或损坏需要记录原因并按内部流程处理。资产台账不包含销售报价、技术密钥或人事档案。
