# 会议室预约规范（admin-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构行政部内部资料，只允许行政部和管理员访问。本文用于回答“会议室如何预约”等问题。

## 2. 预约流程
会议室预约由申请人在系统提交时间、人数、会议主题、设备需求和是否有访客。行政部或系统检查容量、时间冲突和设备可用性。涉及客户接待、跨部门评审或高优先级项目时，可按优先级协调会议室。

## 3. 使用规则
会议组织者应按预约时间使用会议室，不得长期占用或私自转让。需要投影、视频会议、演示机器人或访客接待支持时，应提前标注。会议结束后需恢复桌椅、关闭设备、释放房间，并报告异常。

## 4. 访客与安全
有外部访客参加会议时，行政部需要核验预约信息和来访登记。访客不得进入未授权办公区域，不得拍摄内部系统页面。会议材料如包含内部流程、报价、技术日志或员工信息，必须确认参与人权限。

## 5. 可回答的问题
- 会议室如何预约？
- 会议室冲突如何处理？
- 访客会议需要注意什么？
- 会议结束后要做什么？
- 会议材料能不能包含内部敏感资料？
