# 售后工单 SOP（support-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构客服部内部资料，只允许客服部和管理员访问。本文用于回答“客服部售后处理流程是什么”“工单如何分级”等问题。

## 2. 售后处理流程
客服部售后处理流程为：工单接收、信息补齐、优先级分级、一线处理、必要时转技术部、客户同步、恢复验证、回访关闭。客服专员需记录设备编号、故障现象、影响范围、发生时间和客户期望，不得在口头沟通中跳过工单留痕。

## 3. 工单分级
P1 工单表示核心业务中断或多设备不可用，需要立即响应并升级服务经理。P2 工单表示主要功能受限但仍可降级运行，需要当日跟进。P3 工单表示低影响问题、咨询类问题或可排期处理事项。分级应以影响范围和业务紧急度为准。

## 4. 转技术部条件
机器人常见故障中，网络持续异常、传感器误报、电机状态异常、导航定位失败、任务队列异常、SDK/API 调用错误等需要转技术部。客服部负责补齐现象和客户影响，技术部负责日志、遥测和模块诊断。
