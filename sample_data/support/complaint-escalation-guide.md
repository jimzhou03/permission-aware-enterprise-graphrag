# 投诉升级指南（support-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构客服部内部资料，只允许客服部和管理员访问。本文用于回答“客户投诉如何升级”等问题。

## 2. 投诉识别
客服部将投诉分为服务时效投诉、故障重复投诉、交付范围争议、沟通体验投诉和重大业务影响投诉。客服专员先确认客户诉求、影响范围、已有工单、当前处理人和承诺时间，避免遗漏上下文。

## 3. 升级路径
普通投诉由客服专员和服务经理在工单内处理。涉及重大业务影响、媒体风险、合同争议或跨部门责任不清的投诉，应升级服务经理并拉入销售、技术、产品或管理层。升级时必须附上时间线、证据、当前结论和待决事项。

## 4. 回访与复盘
投诉处理后，客服部需要进行客户回访，确认问题是否关闭、客户是否接受方案、是否需要后续观察。复盘记录包含根因、流程缺口、预防动作和责任部门，但不得把客户隐私或工单明细公开给无关部门。

## 5. 可回答的问题
- 客户投诉如何升级？
- 投诉升级需要哪些材料？
- 哪些投诉需要服务经理介入？
- 投诉处理后如何回访？
- 投诉工单能不能公开给其他部门？
