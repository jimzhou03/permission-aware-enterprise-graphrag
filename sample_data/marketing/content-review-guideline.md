# 内容发布审核规范（marketing-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构市场部内部资料，只允许市场部和管理员访问。本文用于回答“内容发布审核规范是什么”“对外宣传材料需要注意什么”等问题。

## 2. 审核流程
市场部内容发布前必须完成事实核对、权限边界检查、品牌口径检查和发布渠道确认。内容运营提交初稿，品牌经理检查表达一致性，相关部门确认公开范围，最后由市场负责人批准发布。涉及产品能力、售后承诺或商务价格时，需要对应部门复核。

## 3. 内容禁区
内容发布不得包含内部报价策略、折扣审批、真实客户名单、工单明细、员工资料、接口密钥、设备凭据、未发布路线图和内部测试日志。客户案例必须脱敏，并确认授权范围。不能把“规划中”写成“已上线”。

## 4. 发布后复盘
发布后市场部记录阅读量、线索转化、受众反馈、常见误解和纠错记录。若发现内容误导或越界，应立即下线并修正，同时在内部复盘中标明原因和责任人。
