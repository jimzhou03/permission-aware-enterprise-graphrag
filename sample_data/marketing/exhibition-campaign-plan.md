# 展会宣传方案（marketing-internal）

## 1. 适用范围
本文件属于星海智造机器人有限公司 Demo v0.9.4 的虚构市场部内部资料，只允许市场部和管理员访问。本文用于回答“展会宣传方案是什么”“市场活动如何评估”等问题。

## 2. 展会目标
市场部展会宣传方案以品牌曝光、线索收集、场景教育和生态合作为主要目标。展会主题围绕服务机器人、巡检机器人、迎宾机器人、配送机器人四类公开产品线展开，但现场脚本不得披露内部报价、真实客户名单或未发布路线图。

## 3. 物料准备
展会物料包括主视觉展板、产品单页、场景演示视频、公开 FAQ 卡片、线索登记表和演示话术。市场部负责内容口径，销售部负责商务线索跟进，技术部只在授权范围内解释公开技术能力。所有物料发布前必须通过内容审核。

## 4. 活动评估
市场活动评估指标包括有效线索数、目标受众匹配度、现场问题类型、内容领取量、后续转化状态和复盘问题清单。评估不使用客户隐私明细作为公开展示材料，只在内部授权范围内用于优化下一次活动。
